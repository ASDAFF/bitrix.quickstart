# Release Notes for epages.pickpoint module

### 2.1.4 (01.12.2016)
Исправлены ошибки связанные с шаблоном оформления заказа по умолчанию.

### 2.1.3 (24.10.2016)
Исправление определения городов в областях.

### 2.1.2 (23.09.2016)
* Исправление отображения службы доставки для городов, которые битрикс вынес из областей.
* Исправление колизии языковой фразы названия модуля с названием другого модуля.
* Исправление добавления заказа в "Pick Point Экспорт".
* Исправление закрития окна выбора постаматов после выбора постамата.

### 2.1.1 (10.08.2016)
Актуализирован список городов отправки в настройках тарифных зон

### 2.1.0 (29.06.2016)
Отображение службы доставки только для городов в которые осуществляется доставка.

### 2.0.7 (02.06.2016)
Исправлена ошибка работы модуля, связанная с выходом 1С-Битрикс v16, когда заказы, оформленные с доставкой PickPoint, не попадали на страницу экспорта.

### 2.0.6 (10.03.2016)
Исправлена работа блока выбора постамата. Изменен логотип.

### 2.0.5 (05.02.2016)
Исправление кодировки скриптов.

### 2.0.4 (03.02.2016)
Исправлена небольшая ошибка при подключении JS файлов виджета PickPoint в ajax-шаблонах оформления заказов.

### 2.0.3 (05.01.2016)
* Добавлена поддержка новой версии модуля Интернет-магазин после конвертации.
* Исправлена ошибка при сохранении настроек модуля в кирилице для сайтов в кодировке windows-1251.
* Исправлена работа в режиме обеъединения и сжатия JavaScript.
* Исправлена ошибка, когда после измения службы доставки заказ оставался в списке на экспорт в PickPoint.
* Добавлена настройка для указания цены заказа, от которой доставка будет считаться бесплатной.

### 2.0.2 (20.01.2015)
Незначительные изменения на странице настроек модуля

### 2.0.1 (14.10.2014)
Исправления логики калькулятора доставки.
