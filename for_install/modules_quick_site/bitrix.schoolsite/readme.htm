1C-Битрикс: Сайт школы, базовый.

Готовое решение, которое включает в себя 3 шаблона дизайна и адаптированную структуру сайта образовательного учреждения.

Шаблоны и структура сайта соответствует решению "1С-Битрикс: Сайт школы" http://www.1c-bitrix.ru/solutions/edu/school/.

В данном решении не предусмотрен следующий функционал: дневники, журналы, расписание, группы пользователей учеников, родителей и учителей, нет социальной сети и рабочих групп.

Решение может быть установлено на любую редакцию, начиная со Старта.
Для установки решения требуется наличие активной технической поддержки.

Скачать отдельным дистрибутивом можно на сайте: http://www.1c-bitrix.ru/download/edu/school.php (на базе редакции Старт)
