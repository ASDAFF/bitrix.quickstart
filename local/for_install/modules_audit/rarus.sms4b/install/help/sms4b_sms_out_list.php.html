Отправленные SMS

Обратно к содержанию

Показывает список исходящих SMS-сообщений.

Фильтр

Задание параметров отбора исходящих SMS-сообщений, которые отображаются в списке.

Поле Описание
ID Идентификатор сообщения.
GUID Глобальный идентификатор сообщения.
Имя отправителя Имя отправителя сообщения.
Имя получателя Имя получателя сообщения.
Количество частей Количество частей в сообщении.
Частей отправлено Сколько частей сообщения отправлено.
Тип кодировки сообщения Каким образом закодировано сообщение.
ID заказа Идентификатор, указывающий на привязку сообщения к заказу в Интернет-магазине.
ID рассылки Идентификатор рассылки, указывающий на привязку к какой-либо рассылке.
Событие Событие, вызвавшее данное SMS-сообщение.

Чтобы применить фильтр, необходимо нажать кнопку Найти. Для снятия фильтра - кнопку Отменить.

Вспомогательная форма Действия позволяет осуществить различные операции одновременно над несколькими SMS. Чтобы применить то или иное действие к группам, следует пометить их галочками в левом столбце таблицы, а затем выбрать нужное действие.
