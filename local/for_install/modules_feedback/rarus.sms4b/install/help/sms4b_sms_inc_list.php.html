Показывает список входящих SMS.

Фильтр

Задание параметров отбора исходящих SMS-сообщений, которые отображаются в списке.

Поле Описание
GUID Глобальный идентификатор сообщения.
Отправитель Имя отправителя сообщения.
Получатель Имя получателя сообщения.
Всего частей Количество частей в SMS-сообщении.

Чтобы применить фильтр, необходимо нажать кнопку Найти. Для снятия фильтра - кнопку Отменить.

Таблица Входящих сообщений

Поле Описание
Доставлено Дата получения сообщения.
Отправитель Числовой или символьный номер отправителя.
Получатель Числовой или символьный номер получателя.
Текст сообщения Текст полученного сообщения.
Всего частей Количество частей в сообщении (если сообщение состоит из нескольких частей).
