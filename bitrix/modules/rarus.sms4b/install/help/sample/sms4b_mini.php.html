Включение AWARENESS API для мини-подписки

Обратно к содержанию

Для корректного подсчета количества RSS-подписчиков компонентом мини-подписки необходимо:

  1. Авторизоваться на сайте www.feedburner.com;
  2. Выбрать нужный поток;
  3. Перейти во вкладку PUBLICIZE -> Awareness API;
  4. Нажать Включить.
