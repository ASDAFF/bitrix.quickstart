Компонент подписки

Обратно к содержанию

Описание компонента subscribe.index. Форма подписки на рассылку.

Пример вызова

$APPLICATION->IncludeComponent("sms4b:subscribe.index", ".default", Array(
	"SHOW_COUNT"		=>	"N",
	"SHOW_HIDDEN"		=>	"N",
	"SHOW_POST_FORM"	=>	"Y",
	"SHOW_SMS_FORM"	=>	"Y",
	"SHOW_RUBS"		=>	array(
		0	=>	"5",
		1	=>	"7",
		2	=>	"3",
		3	=>	"6",
		4	=>	"4",
		5	=>	"1",
		6	=>	"8",
		7	=>	"2",
	),
	"PAGE"		=>	"#SITE_DIR#dev/egza/subscribe_edit.php",
	"CACHE_TYPE"	=>	"N",
	"CACHE_TIME"	=>	"3600",
	"SET_TITLE"	=>	"Y"
	)
);

Описание параметров

SHOW_COUNT
Если опция установлена, то в публичной части будет выведено количество подписчиков на представленные рассылки.
SHOW_HIDDEN
При установленной опции в списке рассылок будут выведены все активные рассылки вне зависимости от опции Выводить в списке публичных рассылок в настройках рассылки.
SHOW_POST_FORM
При установленной опции будет отображаться форма для оформления подписки по почте.
SHOW_SMS_FORM
При установленной опции будет отображаться форма для оформления подписки по SMS.
SHOW_RUBS
Параметр задает номера рубрик, на которые возможна подписка. Если этот массив пустой, то подписка возможна на все имеющиеся рубрики. Если выбрана хотя бы одна рубрика, то будет возможно осуществить подписку только на выбранные рубрики.
PAGE
Указывается путь к странице редактирования подписки, т.е. странице, на которой располагается компонента subscribe.edit.
CACHE_TYPE
Тип кеширования:
  • A - Авто: действует при включенном кешировании в течение заданного времени;
  • Y - Кешировать: необходимо определить время кеширования;
  • N - Не кешировать: кеширования нет в любом случае.
CACHE_TIME
Время кеширования, указанное в секундах.
SET_TITLE
При отмеченной опции в качестве заголовка страницы будет установлена строка "Рассылки".
