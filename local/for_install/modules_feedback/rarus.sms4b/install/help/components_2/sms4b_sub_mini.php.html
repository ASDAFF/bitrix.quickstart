Компонент мини-подписки

Обратно к содержанию

Компонента для компактного отображения ссылок на страницы подписки на рассылки различного типа (почтовые, SMS, RSS).

Пример вызова

$APPLICATION->IncludeComponent("sms4b:subscribe.minisub", ".default", Array(
	"PAGE"				=>	"#SITE_DIR#dev/egza/index.php",
	"SHOW_POST_SUB"			=>	"Y",
	"SHOW_RSS_SUB"			=>	"Y",
	"URL_FOR_FEEDBURNER"		=>	"http://feeds.feedburner.com/",
	"FEED_NAME"			=>	"retaildotru",
	"SHOW_SMS_SUB"			=>	"Y"
	)
);

Описание параметров

PAGE
Путь к странице с регистрацией подписок.
SHOW_POST_SUB
При выбранной опции отображается ссылка на подписку по почте.
SHOW_RSS_SUB
При выбранной опции отображается ссылка на подписку по RSS.
SHOW_SMS_SUB
При выбранной опции отображается ссылка на подписку по SMS.
URL_FOR_FEEDBURNER
Адрес сайта FEEDBURNER.
FEED_NAME
Имя зарегистрированного потока на сайте FEEDBURNER'a.

Для корректной работы подсчета RSS-подписчиков, необходимо на сайте FEEDBURNER в настройках потока включить поддержку AWARENESS API. В противном случае подсчет RSS подписчиков производиться не будет. Подробнее
