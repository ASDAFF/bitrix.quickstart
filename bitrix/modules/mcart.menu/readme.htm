Установка
1. Устанавливаем модуль на сайт.

2. Для использования возможностей модуля разместите компонент mcart.menu на своей странице:

Шаблон top необходимо размещать после headera
  • Откройте файл шаблона сайта (Битрикс24) и найдите, где заканчивается header.
  • Подключите компонент сразу после header
в этом месте: 
<div id="header">
      ..код headerа..
</div> 
!!сразу здесь подключайте!!
с помощью следующего кода: 
<?$APPLICATION->IncludeComponent(
   "mcart:menu",
   "top",
   Array(
      "ROOT_MENU_TYPE" => "top",
      "MAX_LEVEL" => "2",
      "CHILD_MENU_TYPE" => "left",
      "USE_EXT" => "Y",
      "DELAY" => "N",
      "ALLOW_MULTI_SELECT" => "N",
      "MENU_CACHE_TYPE" => "N",
      "MENU_CACHE_TIME" => "3600",
      "MENU_CACHE_USE_GROUPS" => "Y",
      "MENU_CACHE_GET_VARS" => array()
   )
);?>

Шаблон bottom размещайте в визуальном редакторе на любой необходимой странице, в header.php или footer.php. Разместить можно в любом месте на странице.

Для отображения меню, желательно выполнить следующие настройки:
- тип меню для первого уровня: Главное меню
- Уровень вложенности меню: 2
- тип меню для остальных уровней: Меню раздела
- подключать файлы с именами вида .тип_меню.menu_ext.php: отметить выбранным
