# Заказ обратного звонка

Описание

Внимание! 
Выпущено решение "Заказ обратного звонка Pro". Это аналог данного решения, но не использует для хранения инфоблоки и более простой в настройке. 
Найти решение можно по адресу: 
http://marketplace.1c-bitrix.ru/solutions/wsm.callbackpro/ 

Станьте ближе к Вашим клиентам, используя форму заказа обратного звонка, с уведомлением о заказе звонка на email и мобильный телефон. Быстрая установка и настройка формы - готовность к работе за несколько простых шагов. 

Гибкость настройки позволит оптимизировать модуль под любой бизнес. Будьте привлекательнее для Ваших клиентов - выбирайте лучшее. 

Возможности: 

- уведомление на разные Email и телефон в зависимости от выбранной темы звонка 
- возможность задать дополнительные поля для заполнения 
- настройка выводимых в форму полей 
- подключение SMS модуля 
- хранение заказов звонка в инфоблоке (это позволит при необходимости расширить функционал через события и использовать документооборот - преимущество перед хранением данных в таблицах модуля!) 
- отправки формы заказа звонка без перезагрузки страницы 
- поддержка многосайтовости
