Компонент Фильтр элементов инфоблока выводит форму фильтрации на сайте и расширяет возможности стандартного:

- Добавлены параметры расширяющие функционал компонента.
- В комплекте 4 шаблона и 25 тем для ползунков.
- Есть автокомплит значений свойств типа текст
- Добавлена возможность фильтрации по Свойству привязки к элементам инфоблока в виде списка, работает и для множественных свойств списков, в стандартном компоненте нет такой возможности, выводится обычное текстовое поле.
- Добавлена возможность фильтровать элементы по свойству Привязка к разделам

Обратите внимание!
1. Если на вашем сайте выключено кеширование, то каждое поле интервал с ползунком + 2 запроса к базе данных.
Для автокомплита строковых свойств - всего 1 запрос к БД.

Если включено кеширование - запросов к базе данных 0;


2) Если включаете библиотеку jQuery UI, то библиотеку jQuery лучше отключить, это чуть увеличит скорость загрузки скриптов, т.к. jQuery уже есть в jQuery UI

3) Если в настройках компонента включена опция проверка активности разделов и в указанном инфоблоке нет разделов, то поля интервалы будут пустые.

Подписаться на обновления всех моих модулей Вы можете на моем сайте в форме в самом низу.

Как сделать фильтрацию в комплексном компоненте каталога?
1) Размещаем компонент Фильтр по элементам в визуальном редакторе
http://joxi.ru/H220Uv3JTJAPVGqZoZI

2) Переключаем виз. редактор в Режим редактирования исходного кода, выделяем код компонента и копируем в буфер Ctrl + C
http://joxi.ru/zm20Uv3JTJDpU7VlOLY

3) Переходим к странице на которой размещен комплексный компонент каталога и в его настройках включаем опцию Показывать фильтр, больше ничего не трогаем, нажимаем Сохранить
http://joxi.ru/TnO0UhjKTJCcGBC7Mbc

4) Открываем шаблон комплексного компонента Каталога для редактирования
http://joxi.ru/W260Uv3JTJDiU5vcSLc

5) Находим код вызова компонента bitrix:catalog.smart.filter
http://joxi.ru/TXK0UhjKTJDHGJIlaJI

6) Заменяем на код компонента api:search.filter
http://joxi.ru/1XK0UhjKTJCKGGWvNCM

7) Нажимаем сохранить


Теперь настроем компонент api:search.filter


Открываем настройки компонента api:search.filter
http://joxi.ru/UXS0UhjKTJCkGLdHti0

9) Выбираем шаблон, например vertical-dark-red, задаем необходимые для фильтрации свойства, с другими настройками поиграйтесь потом, пока проверим работу фильтра и нажимаем сохранить.

10) Все, проверяйте работу фильтра, на стандартном компоненте каталога фильтр должен работать.

Как сделать фильтрацию в компоненте каталога - Элементы раздела?
В случае с простым компонентом bitrix:catalog.section фильтр работает без всяких манипуляций - компонент фильтра достаточно разместить на странице над компонентом bitrix:catalog.section

Обратите внимание!!!
Если фильтр и каталог Вы разместите на разных страницах сайта, то фильтр работать не будет.
Фильтр должен всегда находиться на странице с каталогом или в шаблоне сайта.
