ОСНОВНЫЕ КОМПОНЕНТЫ, ИХ ФУНКЦИОНАЛ И ВОЗМОЖНОСТИ

Каталог. Удобный двухуровневый каталог товаров, уровни меню по категориям и по производителю (верхнее и левое меню сайта); "Умный фильтр", динамическая фильтрация в списке товаров по любым свойствам/набору характеристик; Группировка по области использования (частные клиенты, бизнес, …); Объединение товаров в списки по принадлежности к редакции или группе.

Товар. Вывод описаний: краткое описание и основные свойства товара, расширенное описание, дополнительное описание товара во вкладках, общие элементы описания для нескольких товаров сразу, вывод товаров из категории списка товаров, вывод похожих товаров. Изображение: добавление несколько изображений к товару, увеличение изображения; автоматическое создание превью (уменьшенных изображений товара),

Оформление заказа. Функция быстрого заказа со страницы товара “Заказ в один клик”; “Пошаговый мастер оформления заказа”, Возможность оформления заказа без регистрации (покупка возможна только при указании электронного адреса, логин и пароль система создаст автоматически); Система уведомления о неоплаченных заказах; Работа с физ. и юр.лицами.
