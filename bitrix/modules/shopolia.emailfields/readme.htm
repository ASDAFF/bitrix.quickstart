Установить
Добавляет в стандартные почтовые шаблоны значения свойств заказа. Применимо для писем с уведомлением об оформлении нового заказа, о разрешении доставки.
Подробнее
Скриншоты Поддержка Установка
Скриншоты
Поддержка
Для бесплатного решения техподдержка не предлагается.
Установка
Устанавливается как обычный модуль, не имеет настроек. После установки модуля в шаблоне для типа события SALE_NEW_ORDER (оформление нового заказа) появляются новые доступные переменные, содержащие значения полей заказа. Например, если у заказов есть поле с кодом FIO, содержащее значение имени заказчика, то в шаблоне письма становится возможным использовать новые переменные:
#PROP_FIO# - вывод названия и значения поля с кодом FIO, в письме появится строка вида "Имя заказчика: Иванов Иван"
#PROP_VALUE_FIO# - вывод только значения поля с кодом FIO
#PROP_NAME_FIO# - вывод только названия поля с кодом FIO
Можно использовать поля всех типов, для которых задан мнемонический код.

Для типа события SALE_ORDER_DELIVERY (доставка разрешена) доступны поля:
#DELIVERY_DOC_NUM# - номер документа о доставке, который указывается в заказе
#DATE_ALLOW_DELIVERY# - дата документа о доставке
