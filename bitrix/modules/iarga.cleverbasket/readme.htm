Эта корзина подстраивается под любые мобильные устройства, сохраняет введённые пользователей данные, определяет город, из которого делается заказ и при этом - неплохо выглядит.
Мы создали её для использования в собственных сайтах и с радостью предлагаем попользоваться вам.

Сейчас корзина поддерживает автоматизированные и ручные расчёты доставки, скидки по купонам.

Язык пока только русский. Авторы будут благодарны за помощь в переводе на любые известные пользователям языки.

После этого В компоненте корзины bitrix:sale.basket.basket появятся 2 новых шаблона корзины.

iarga.cleverbasket - для сайтов, где уже используется jQuery
iarga.cleverbasket_nojq - для сайтов, где jQuery Не используется

Если у одного из свойств заказа код содержит слово phone, то это свойство будет проверяться как телефон (на соответствие формату +79030000000)

Если у вас нет свойства типа Location, то служба доставки не сохранится в заказе.
