Компоненты 2.0

Обратно к содержанию

В стандартный дистрибутив модуля входит 3 компонента:

  • subscribe.index (Форма подписки)

    • Обычно используется в паре с компонентом редактирования подписки subscribe.edit.
    Обеспечивает:

    • создание новой подписки;
    • выбор рубрик новой подписки;
    • отмена подписки (для зарегистрированных пользователей);
    • переход к редактированию подписки.
  • subscribe.edit (Форма редактирования подписки)

    • Обычно используется в паре с компонентом формы подписки subscribe.index.
    Обеспечивает:

    • создание новой подписки;
    • редактирование подписки;
    • отмена подписки.
  • subscribe.minisub (Компонент мини-подписки)

    • Компонент мини-подписки служит для перехода к оформлению различного рода подписок (RSS-,SMS-, почтовых рассылок). Также включает в себя информацию о количестве подписчиков на соответствующие типы рассылок.

Термины

  • Залогинненый пользователь

    • Пользователь, который ввел логин и пароль от сайта (осуществил запуск сессии). Эта процедура требуется для того, чтобы опознать посетителя сайта (зарегистрированный на сайте).

  • Анонимный пользователь (далее аноним или анонимный подписчик)

    • Не прошедший авторизацию пользователь (не зарегистрированный на сайте).

  • Рассылки

    • Тематические рубрики, на которые могут подписаться посетители сайта. Возможна одновременная подписка на несколько рассылок.

  • Подписчики

    • Посетители сайта, подписавшиеся на рассылки.
