Настройки SMS-модуля

Обратно к содержанию

Для задания параметров модуля SMS4b служит форма Настройки (Настройки > Настройки продукта > Настройки модулей > SMS4b).

В данной форме имеется две вкладки:

  • Настройка. Общие параметры модуля
    • Параметр Описание
    Адрес SMS4b-сервера Адрес сервера для отправки SMS-сообщений (https://sms4b.ru).
    Адрес прокси-сервера Адрес прокси-сервера, через который будет происходить подключение.
    Порт прокси-сервера Порт прокси-сервера, через который будет происходить подключение.
    Использовать прокси-сервер Нужно ли вообще использовать прокси-сервер.
    Логин пользователя SMS4b Логин учетной записи на SMS4b-сервере.
    Пароль пользователя SMS4b Пароль учетной записи на SMS4b-сервере.
    Часовой пояс Этот параметр позволяет выбрать текущий часовой пояс. Он служит для синхронизации времени отправки и доставки SMS-сообщений.
    Отправитель по умолчанию У пользователя SMS4b-сервера может быть несколько имен в одной учетной записи. В теле SMS-сообщения в поле Отправитель будет указано значение данного параметра.
    Максимальная длина текста SMS Ограничение на количество символов в SMS-сообщении. Если значение не задано, ограничение отсутствует.
    Автотранслит SMS-рассылок Автоматическая транслитерация текста сообщения из кириллицы в латиницу. Включение этого параметра позволяет увеличить количество символов для отправки в одном SMS-сообщении.

  • Почтовые события. Позволяет настроить почтовые события, обрабатываемые SMS-сервисом.
    • Почтовое событие Описание
    Подтверждение подписки При регистрации подписки приходит SMS-сообщение с кодом подтверждения подписки. Если эта опция включена, пользователю придет уведомление о подтверждении подписки.
    Изменение статуса заказа При смене статуса заказа пользователю будет отсылаться уведомление.
    Подписка отменена При отмене подписки пользователю будет высылаться уведомление.
    Заказ оплачен Уведомление об оплате заказа.
    Доставка заказа разрешена Уведомление о разрешении доставки заказа заказчику.
    Отмена заказа Уведомление об отмене заказа.
    Новый заказ Уведомление о новом заказе.

Почтовые сообщения приходят пользователям Интернет-магазина и подписчикам.
