Добавление SMS-подписчика

Обратно к содержанию

Процесс создания SMS подписки практически ничем не отличается от создания почтового подписчика, но необходимо учитывать следующие детали:

  1. При указании телефонного номера, в поле EMAIL , необходимо добавить постфикс @phone.sms (Это связано с особенностью реализации системы);
  2. Должен быть указан валидный телефонный номер.
Пример правильного значения поля EMAIL - 79031255567@phone.sms
