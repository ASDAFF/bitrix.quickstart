Описание модуля

Модуль SMS4b позволяет расширить функционал сайта посредством следующих возможностей:

  • при совместном использовании с модулем «Подписка и рассылки» организовывать рассылки выпусков посредством SMS-сообщений;
  • совместно с модулем «Интернет-магазин» оповещать пользователя о таких событиях, как:
    • создание заказа;
    • изменение статуса;
    • отмена заказа.
  • отслеживать обращения клиентов посредством SMS;
  • отсылать сообщения в публичной и административной части сайта.

Перед началом использования обязательно ознакомьтесь с логикой работы и особенностями модуля.

Требования

  • PHP 5 или выше, модуль curl;
  • MYSQL 4.1 или выше.
